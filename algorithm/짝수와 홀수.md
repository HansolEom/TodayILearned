[짝수와 홀수](https://programmers.co.kr/learn/courses/30/lessons/12937?language=javascript)

### 입출력

| num | return |
| --- | --- |
| 3 | “Odd” |
| 4 | “Even” |

### 풀이

```jsx
function solution(num) {
    return answer = num%2==0 ? "Even" : "Odd"
}
```

조건부 연산자를 사용하여 num을 2로 나눈 나머지가 0이면 Even, 아니면 Odd를 return한다.
